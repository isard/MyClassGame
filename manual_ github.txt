Instal·lació màquines:
----------------------
sudo apt-get install git
git init
git remote add origin <server>


1a Màquina:
-----------
git add .
git commit -m "Commit message"
git pull
git push -u origin master

2a Màquina:
-----------
git pull origin master

Forzar git:
-----------
git fetch --all
git reset --hard origin/master
